Question:
- If you create a constructor method to build a DVD by title then create a constructor method to build
a DVD by category. Does JAVA allow you to do this?
Answer:
Java không cho phép điều này vì title và category đều là kiểu String và cùng có 1 tham số đầu vào như vậy sẽ khiến cho
Java không xác định là truyền cho cái nào, vì thế chữ ký là cần số lượng tham số khác nhau hoặc có kiểu dữ liệu khác nhau
What do you prefer in this case?
Answer:
Em ưu tiên sử dụng cho phép truyền vào lượng tham số tùy ý vì linh hoạt và code gọn hơn. Nếu dùng mảng thì ở main phải
khai báo trước 1 mảng rồi mới truyền vào.
